# Anvil-v7M
This is an emulator of an ARM-M3 microprocessor

## Datasheet

- 1x Core ARMv7-M 32 bits @ 16mhz

## Memory map
| DIRECTION | NAME | SIZE |
|:----------|:------:|--------:|
| 0x00000000 — 0x00003FFF     | Internal FLASH  | 16 KB   |
| 0x20000000 — 0x20001FFF | Internal SRAM    | 8 KB     |
